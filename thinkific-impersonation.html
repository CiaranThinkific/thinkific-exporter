Private window This page cannot open an incognito tab for you. Use Copy SSO link and paste the URL into a private window. Open in new tab uses your normal browser profile.

CORS This runs in the browser. If GraphQL fails with CORS, use a local server or CLI. Clipboard copy may require HTTPS or localhost.

Storage Saved keys live in this browser only. If the device is compromised, treat secrets as exposed.

Cache After a full load, the user list is saved for this endpoint + API key + SSO secret. If those match when you open the page, the list restores automatically. Use Load users to refresh from the API. Re-save secrets if you had an older cache without a secret fingerprint (run Load users once).
