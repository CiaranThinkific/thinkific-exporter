Enrollments + instructor roster

Loads every enrollment via the REST API (paginated, rate-limit aware), then loads courses and instructors from GraphQL (stable). Rows are merged on course id and exported as CSV.

CORS Runs in the browser. If you see CORS errors, use a small local proxy or CLI script. REST throttling uses X-RateLimit-Remaining-Minute and RateLimit-Reset when present, plus 429 / Retry-After.
